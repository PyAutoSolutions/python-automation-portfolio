# Web Scraper con Python

Script que extrae títulos y precios de libros de una página web.

## Tecnologías
- Python
- requests
- BeautifulSoup4

## Uso
1. Instalar: `pip install requests beautifulsoup4`
2. Ejecutar: `python web_scraper.py`

## Resultado
Muestra los primeros 5 libros con título y precio.
